# 🏥 AlHuda Med — منصة إدارة مطالبات التأمين الطبي

نموذج أولي (Prototype) لمنصة سحابية داخلية لتحليل وتتبع حالات التأمين الطبي وأخطاء نموذج اليوكاف (UCAAF).

---

## التشغيل السريع

```bash
pip install -r requirements.txt
python data/seed.py          # توليد بيانات تجريبية (مرة واحدة)
streamlit run main.py
```

ثم افتح المتصفح على: **http://localhost:8501**

## هيكل المشروع

```
alhuda-med/
├── main.py                    ← نقطة الدخول الرئيسية (Streamlit)
├── requirements.txt
├── data/
│   ├── seed.py                ← مولّد البيانات التجريبية
│   └── claims.csv             ← يُنشأ تلقائياً
└── app/
    ├── auth.py                ← نظام تسجيل الدخول
    ├── ucaaf_analyzer.py      ← محرك تحليل أخطاء اليوكاف
    ├── components.py          ← مكونات الرسوم البيانية (Plotly)
    └── pages/
        ├── doctor_dashboard.py  ← لوحة الطبيب
        ├── admin_dashboard.py   ← لوحة الإدارة
        └── ucaaf_upload.py      ← رفع وتحليل اليوكاف
```
